# Diretrizes editoriais

## Limites de campo

- Hero eyebrow: 55 caracteres.
- Hero título: 92 caracteres.
- Hero subtítulo: 240 caracteres.
- CTA: 42 caracteres.
- Título de produto: 55 caracteres.
- Descrição de produto: 190 caracteres.
- Benefício: 90 caracteres.
- Pergunta FAQ: 120 caracteres.
- Resposta FAQ: 600 caracteres.

## Regras

- Não publicar HTML bruto.
- Não usar links vazios, `javascript:` ou URLs locais.
- Não prometer garantia absoluta de resultado, exclusividade ou completude de dados.
- Validar desktop e mobile antes de publicar.
