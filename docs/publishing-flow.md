# Fluxo de publicação

1. Editar conteúdo.
2. Salvar rascunho.
3. Revisar preview desktop, tablet e mobile.
4. Publicar com confirmação.
5. Registrar revisão e publicação.
6. Revalidar rotas afetadas.
7. Conferir página pública.

Rotas revalidadas por padrão: `/`, `/produtos`, `/montar-minha-base`, `/para-quem-e`, `/como-funciona`, `/faq`, `/contato` e `/sobre`.
